## Coding task optimization
 Оптимизация кода - это преобразование кода в "оптимальный код" с меньшем потреблением ресурсов (памяти) и\\или более быстрым временем выполнением.
 Оптимизация может выполнять компилятор, в этом случае 
 + результат должен быть корректным, программа на выходе никак не должна поменяться 
 + программа должна стать более быстрой и с лучшей производительностью
 + должна быть приемлемое время компиляции

Когда оптимизировать - обычно в конце разработки, так как оптимизация иногда снижает читаемость и может давать некоторый излишний код для оптимальности (например, кеширование)

Почему - оптимизация может уменьшить потребляемую память, повысить скорость работы, скорость компиляции, повысить переиспользованость. 

Типы оптимизаций:
1. **Машинно-независимая оптимизация** - это оптимизация "промежуточного кода", чтобы получить лучше целевой код. Данная оптимизация не заточена на какую-то платформу
2. **Машинно-зависимая оптимизация** - оптимизация целевого кода под конкретную архитектуру/платформу.  

Оптимизации могут быть выполненными как пример следующими путями: 
1. Оптимизация за счет вычисления значений во время компиляции:
```C++
auto res = 2 * (22.0 / 7.0) * len;
// vs 
auto multiplier = 2 * (22.0 / 7.0);
auto res = multiplier * len;
```
2. Variable Propagation (распределение переменных) - избавление от лишних переменных, оптимизация переиспользования переменных
```c++
    int x = fun(); 
    int y = x; 
    int z = x + y + 2; 
    // vs 
    int x = 3; 
    int y = 3; 
    int z = x + y + 2; 
```
3. Constant Propagation\\ (распространение констант) - подстановка констатнт где это можно?
```c++
a = 30;
b = 20 - a / 2;
c = b * (30 / a + 2) - a;
// vs
a = 30;
b = 20 - 30 / 2;
c = b * ( 30 / 30 + 2) - 30;
```
4. Constant Folding (развертка констант) - подсчет значений констант где это возможно
```c++
#define k 5
x = 3 * k;
y = k + 5;
// vs 
x = 15;
y = 10;
```
5. Copy Propagation (распространение копирования) - дополнение к распространению констант, сокращение потенциальных копирований
```c++
a = 1 + 2; 
b = a; 
ans = b + 6; 
// vs 
a = 1 + 2;
ans = a + 6;
```
6. Удаление общих подвыражений 
```c++
a = b * c + g;
d = b * c * e;
// vs 
tmp = b * c;
a = tmp + g;
d = tmp * e;
```
7. Удаление мертвого кода (код, которые не используется после своего присвоения )
8. Удаление не достижимого кода 
```c++
int main() 
{
    int num;
	num = 10;
	return 0;
	cout << num; //unreachable code
}
```

10. Перегрузка функций
